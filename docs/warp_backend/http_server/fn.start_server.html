Simple HTTP server that provides mainly 3 REST APIs:

  • GET /help
    • provides hints about how to set query strings
  • GET /reservation
    • provides the reservation name corresponding to the ID provided by the SLURM plugin after a positive reservation process
    • Example: /reservation?workflow_id=abcd&step_id=abcd
  • PUT /notify?workflow_id=abcd&step_id=abcd&status=4
    • update the status of a job/step to a value provided by streamflow. Allowed values are listed in Constants @ crate::yaml_prs_cwl
    • Example: /notify?workflow_id=abcd&step_id=abcd&status=4
